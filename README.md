# SGT
#Proyecto de gestión de transporte escolar (SGT), Es un sistema diseñado para dar solución a la problemática de auditoria que tiene el programa de transporte estudiantil (TRAE).

Dentro de estas deficiencia esta la falta de un mecanismo para validar los estudiantes que ingresan a cada autobús, su chófer y si azafata.

Otro de estos es la problemática donde el padre o tutor no puede localizar a su hijo hasta llegar al destino.
